Scriptable

Build your own reverse-engineering tools in a few lines of Python

import frida
p = frida.attach("skype")
for m in p.enumerate_modules():
  print m.enumerate_exports()

Stalking

Stealthy code tracing without relying on software or hardware breakpoints. Think DTrace in user-space, based on dynamic recompilation, like DynamoRIO and PIN.

Portable

Works on Linux, Mac (32- and 64-bit), Windows (32- and 64-bit), iOS.
