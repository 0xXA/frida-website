Dynamic instrumentation toolkit for developers, reverse-engineers, and security researchers.

Scriptable

Inject your own scripts into black box processes. Edit a script, hit save in your text editor, and instantly see the results. No compilation, deployment, or program restart needed. Hook any function, spy on crypto APIs or trace private application code, no source code needed either! Frida can also do stealthy code tracing without relying on software or hardware breakpoints. Think DTrace in user-space, based on dynamic recompilation, like DynamoRIO and PIN, but fully scriptable.

Portable

Built with portability in mind from day one, Frida works on a lot of different OSes and architectures. Currently this means Windows, macOS, GNU/Linux, iOS, Android, and QNX, spanning architectures from Intel to ARM, ARM64, and MIPS. QNX and MIPS are the most recent additions, thanks to a member of our awesome community. We also have a wide range of language bindings. Install the Node.js bindings from npm, grab a Python package from PyPI, or use Frida through our Swift bindings, .NET bindings, Qt/Qml bindings, or C API.

Free

Frida is and will always be free software. We want to empower the next generation of developer tools, and help people achieve interoperability through reverse engineering. Also, recognizing that a lot of companies roll their own dynamic instrumentation, we think it would be awesome if we could all collaborate on building the best possible toolkit to faciliate this. Hence, our licensing allows using Frida in non-free software. We are proud that NowSecure is using Frida to do fast, deep analysis of mobile apps at scale.
